## Basics
* Insert mode, normal mode
```
i " to enter into insert mode
Esc " to exit the insert mode
```

* To open a file
```
" you can use tabs here
:e file_name
```
If a new file is opened, `[New File]` is displayed in the status bar.

* To save a file
```
:w
```

* To exit vim
```
:q
```

* To forcefully exit vim
```
:q!
```
